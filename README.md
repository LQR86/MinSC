# MinSC - Minimal StarCraft for MCP

MinSC (Minimal StarCraft for MCP) 是一个简化的RTS游戏引擎，专为AI Agent通过Model Context Protocol提供Tools API接口而设计。

## 概述

基于StarCraft经典RTS机制，提供：
- 简化的游戏规则和单位系统
- 实时AI对战环境（1v1→NvsN）
- MCP协议集成的Tools API
- Pygame实现的可视化界面

## 技术架构

- **游戏引擎**: Python + Pygame
- **AI接口**: Model Context Protocol (MCP)
- **数据存储**: 内存数据库 + SQLite持久化
- **可视化**: 实时游戏界面
